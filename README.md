# Personal Website

Build with [vite](https://vitejs.dev), [ReactJS](https://reactjs.org) and [tailwindcss](https://tailwindcss.com)!

## Requirements

  - [NodeJS](https://nodejs.dev)

## Development

Start development server

```
yarn dev
```

## Build

Build the output for publishing, the output will be `dist`.

```
yarn build
```
